# Lab-M1.02---Data-Manipulation-JSON-Handling
Titanic Data Analysis and JSON Export
Author: Marc Tanguy
Description: Analyze Titanic passenger data, engineer features, and export to JSON

Brief explantion:
I created family size, isalone, agegrop, sex and passenger class engineered features.
FamilySize → passengers with small families (2-4) had better survival rates than those alone or with very large families
IsAlone → passengers travelling alone had lower survival rates than those with family
AgeGroup → Children had priority on lifeboats so higher survival rates, while adult males had the lowest
Sex → females survived at much higher rates than males ("women and children first")
Pclass → 1st class passengers had significantly better survival rates than 2nd and especially 3rd class, as they had easier access to lifeboats
